Приветствую!

Окружение:
Node.js 8.2.1+
gulp (npm install gulp -g)

Подготовка проекта:
npm install

Команды:
gulp annotations
// Создает в директории ../annotations/ json-файл со всеми аннотациями,
 для более удобной работы с ними в дальнейшем.

gulp crop
// Вырезает фрагменты из исходных изображений, основываясь на данных об аннотациях,
 кладет фрагменты в директорию ../fragments/

gulp greyscale
// Преобразует вырезанные фрагменты в оттенки серого,
кладет фрагменты в директорию ../fragments_greyscale/

gulp flip
// Отражает полученные фрагменты по горизонтали,
кладет фрагменты в директорию  ../fragments_flip/

gulp normalize
// Выполняет нормализацию фрагментов в директории ../fragments_flip/

gulp gauss
// Добавляет на изображения размытие по методу Гаусса,
кладет фрагменты в директорию ../fragments_noise/
